# Note-taking Web App

A full-stack note-taking app based on the Frontend Mentor Note-taking Web App challenge.

The goal is to build a realistic portfolio project with a React frontend, Express backend, PostgreSQL database, authentication, note management, tags, search, themes, and automated tests.

## Planned approach

This project will be built in small, focused commits and pull requests so each part is easy to understand, test, and explain.

## Planned stack

- React
- Vite
- TypeScript
- Node.js
- Express
- PostgreSQL
- Drizzle ORM
- Vitest
- React Testing Library
- Supertest
- Playwright
