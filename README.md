# Note-taking Web App

A full-stack note-taking application based on the Frontend Mentor Note-taking Web App challenge.

The app will allow users to create, edit, delete, archive, search and organise notes with tags. It will also include authentication, persistent user data, theme preferences and automated tests across the frontend and backend.

## Features

- Create, read, update and delete notes
- Archive and unarchive notes
- Search notes by title, content and tags
- Filter notes by tag
- Choose colour and font themes
- Create an account
- Log in and log out
- Change password
- Reset password
- Save notes to a database
- Keep each user's notes private
